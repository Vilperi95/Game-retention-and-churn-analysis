# Game Retention and Churn Analysis
A simple data analysis project focused on understanding player behavior in mobile games.

## Overview
This project analyzes player retention and churn using gameplay event data. The goal was to understand player behavior and identify key moments where users drop off.

## Key Findings
- Retention drops sharply in the first few days (~82% → ~24% by Day 7)
- ~76% of players churn within the first week
- Most players are active only for a few days
- A small group of highly engaged users remains active longer

## Insights
Most players decide whether to continue playing within the first few days. This makes the early game experience critical. Improving onboarding, early progression, and rewards could significantly improve retention.

## Methods
- Calculated retention using days since install
- Analyzed churn based on last activity
- Examined distribution of active days per user

## Tools
- Python (Pandas, Matplotlib)
